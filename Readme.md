# ✨ Ключевые возможности

### 1. Простой калькулятор (Calculator)

- Выполнение базовых операций: сложение `+`, вычитание `-`, умножение `*`, деление `/`.
- Обработка ошибок (например, защита от деления на ноль).
- Возможность проводить несколько вычислений подряд.

### 2. Угадай число (Random Number Guesser)

- Компьютер загадывает случайное число в заданном диапазоне (от 1 до 100).
- Интерактивные подсказки для игрока ("Больше" / "Меньше").
- Подсчет количества попыток, затраченных на угадывание.

## 🚀 Инструкция по запуску

### Требования

Для работы программы вам понадобится установленный интерпретатор (например, **Python**, **Node.js** или компилятор **C++**, в зависимости от того, на каком языке написан ваш код).

### Склонируйте репозиторий

```bash
git clone https://github.com
cd название-репозитория
```

### Запуск приложения

Запустите главный файл проекта через терминал:

- **Если проект на Python:**
  ```bash
  python main.py
  ```
- **Если проект на JavaScript (Node.js):**
  ```bash
  node index.js
  ```

## 🛠️ Стек технологий

- Основной язык программирования Python latest version
- Встроенные модули для генерации случайных чисел (например, `random` или `Math.random()`) импортирован random
